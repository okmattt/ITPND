HTML

HTML stands for Hypertext Markup Language. HTML is a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.
The tags provide tell the computer how to interpret the text.
The tag <b> </> will make text bold.
Two types of tags are inline and block. I don't fully understand the difference other than to say block tags create a box around text. Span is an inline tag.
Div is a block tag.

You can see the difference above. With <span>, it just kept displaying text in line. With <div>, it created a whole new line, like a box.

Google Chrome has a very cool Developer Tools feature that allows you to view a web pages source code. It even lets you change the source code! In the tool you can see how a particular website was structured and the style by viewing both the HTML and CSS. This. Is. Awesome.

A few final notes on HTML and CSS

  • HTML and CSS are both languages

  • CSS controls the style of a web page
